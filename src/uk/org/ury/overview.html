The URY broadcasting and presentation suite.

This system, which goes under the working title SLUT (Streaming Library of URY Tracks), is intended as a replacement for the current (as of March 2011) system in use at University Radio York, BAPS.

Contents

The system comprises:

  • A server system providing access to station assets through a heavily documented JSON/HTTP API;
  • A client implementation providing a frontend similar to the original BAPS system, using the Server API to access data and sound systems;
  • An array of class modules corresponding to the resources for use by both server and client implementation;
  • Documentation for the above.
